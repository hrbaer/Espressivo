## 8BitDo Pro 2 Bluetooth Gamepad

### Modes

This gamepad includes a switch for different modes.

#### Mode S

This mode pretends to be a Switch controller.
Only wired connection supported?

- Motion sensor is available
- Left and right trigger do not provide a continuous signal

#### Mode A

This mode is recommended for MacOS.

- Motion sensor is not available
- Left and right trigger provide a continuous signal

#### Mode D

Tnis is the recommended mode.

- Allows Bluetooth connection
- Motion sensor is available
- Left and right trigger provide a continuous signal

#### Mode X

Mode conforms to Xbox.

- Does not allow Bluetooth connections
- Motion sensor is not available

## Nintendo Switch Gamepad

- Left and right trigger do not provide a continuous signal

You might need to reconnect the gamepad if the motion sensor seems to be inactive.

## Xbox Gamepad

Device has no motion sensor.

- Provides continuous trigger values
